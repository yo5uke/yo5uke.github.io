library(tidyverse)

# id_muni列に政令指定都市は区のコードが入っているデータ
df <- read_csv("data.csv")
seirei_ward <- read_csv("seirei_ward.csv")

df <- df |> 
  left_join(
    seirei_ward |> 
      select(id_ward2025, id_muni2025), 
    by = c("id_muni" = "id_ward2025")
  ) |> 
  mutate(
    id_muni = if_else(
      !is.na(id_muni2025),  # id_muni2025がNAではない、という条件
      id_muni2025,          # 真ならばid_muni2025を
      id_muni               # 偽ならばid_muniをそのまま入れる
    )
  ) |> 
  # 最後に不要になった列を削除
  select(-id_muni2025)

もともとid_muniという列には市区町村コードが入っていると想定しており、政令指定都市の場合は市のコードではなくそれぞれの区のコードが入っているような状況です。

id_muniとseirei_wardのid_ward2025をキーにして結合させると、政令指定都市の行にだけ、区ではなく市のコードが入った列が追加されます（他の行はNAになります）。

市のコードが入ってる行についてのみ、id_muniをid_muni2025上書きして、それ以外はそのままでよいので残すようにmutate()しています3。

最後にid_muni2025列は不要になるので削除しています。

## サンプルコード

```r
library(tidyverse)

# id_muni列に政令指定都市は区のコードが入っているデータ
df <- read_csv("data.csv")
seirei_ward <- read_csv("seirei_ward.csv")

df <- df |> 
  left_join(
    seirei_ward |> 
      select(id_ward2025, id_muni2025), 
    by = c("id_muni" = "id_ward2025")
  ) |> 
  mutate(
    id_muni = if_else(
      !is.na(id_muni2025),  # id_muni2025がNAではない、という条件
      id_muni2025,          # 真ならばid_muni2025を
      id_muni               # 偽ならばid_muniをそのまま入れる
    )
  ) |> 
  # 最後に不要になった列を削除
  select(-id_muni2025)
```

もともと`id_muni`という列には市区町村コードが入っていると想定しており、政令指定都市の場合は市のコードではなくそれぞれの区のコードが入っているような状況です。

`id_muni`と`seirei_ward`の`id_ward2025`をキーにして結合させると、政令指定都市の行にだけ、区ではなく市のコードが入った列が追加されます（他の行はNAになります）。

市のコードが入ってる行についてのみ、`id_muni`を`id_muni2025`上書きして、それ以外はそのままでよいので残すように`mutate()`しています^[もちろん必ずしも上書きする必要はなく、新しい列を作っても問題ありません。]。

最後に`id_muni2025`列は不要になるので削除しています。
